Remit API

Remit is a simple api that allows you to send emails from your node js application. It uses your Gmail account to send emails. You can use it to send emails to your users, send notifications, send verification emails, etc. It is very easy to use and it is free.

How to use?

To use Remit, you need to generate an API token. You can generate an API token by entering your Gmail credentials on the homepage. You can then use the API token to send emails from your node js application. View the documentation for more information.

What is app password?

App password is a password that you can generate from your Gmail account to allow third party applications to access your Gmail account. You can generate an app password by following the steps below:

  1. Go to your Gmail account
  2. Click on your profile picture
  3. Click on Manage your Google Account
  4. Click on Security
  5. Click on App passwords
  6. Click on Select app and choose Other
  7. Enter a name for the app and click on Generate
  8. Copy the app password and use it to generate an API token
